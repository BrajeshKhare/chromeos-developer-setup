Chromeos-developer-setup

A developer setup guide for Chrome OS (Chromebooks).

Download this project as a .zip file Download this project as a tar.gz file

I've recently switched to Chromebook as my main laptop. I have come across several different possible options available to developers. I have detailed them below.

First things first, turn off OS verification. After you turn off OS verification, all of the below are now open to you as options:

Option 1: Ultra-minimalist Setup

  • Description: Use Chrome OS terminal to access Linux 3.4 environment and install a package manger for minimalist dev box.
  • Time to Complete: 5 - 10 minutes.
  • How:
    • Open developer's console (crosh terminal) by pressing + + t
    • Type in shell
    • You are now able to work with the Linux environment. For development purposes, this probably means Vi & SSH'ing to a remote server.
    • If you are feeling too limited but don't want to install Ubuntu or the like, try Option 2.
    • Recommended: you can install Chromebrew, the Chrome OS package manager.
    • It will provide several useful packages such as make, git, python, ruby, vim, openssl, curl, etc. Full list can be found here.

Option 2: ChrUbuntu Setup

  • Description: Dual boot Ubuntu 12.04 and Chrome OS separately. I have found this to be more tedious to initially setup and more suited for those who prefer to boot into one OS at a time.
  • Time to Complete: 15 - 30 minutes
  • How:
    • Follow this excellent tutorial
    • If you want to install Ubuntu on an SD card follow this tutorial

Option 3: Crouton Setup

  • Description: To me, this is the best option because it involves just one command to install, immediate switching between Chrome OS & Ubuntu. Also allows you to combine Option 1 with this option.
  • Time To Install: 20 - 30 minutes
  • How:
    • Download crouton here via chrome.
    • Open developer's console (crosh terminal) by pressing + + t
    • Type in shell
    • Run the installation
    • For XFCE type in: sudo sh -e ~/Downloads/crouton -t xfce
    • For Unity type in: sudo sh -e ~/Downloads/crouton -t unity
    • Near the end of the installation, you will be asked to pick a username and password. Now you should be able to switch between Chrome OS and Ubuntu seamlessly. To switch to Ubuntu:
    • If you installed unity, run sudo startunity
    • If you installed xkfce, run sudo startxfce4
    • Shortcut for switching OSes for ARM based Chromebook: +++ (F2)
      • Replace the back key (F1) with the forward key (F2) to switch back
    • Shortcut for switching OSes for Intel based Chromebook: + + (F1)
      • Replace the back key (F1) with the forward key (F2) to switch back.

Option 4: The Cloud "Koder" Setup

Description: This option relies on using an online service koding.com that hosts free VMs for you which you can access via a very nice web interface and a chrome app called terminal. I plan on writing a short blog post on this setup soon. Time: 5 - 10 Minutes How: Sign up at koding.com for a free account. You will get an Ubuntu virtual machine and a sub-domain name. Install the terminal chrome app. Once installed, open it. It will allow you to access your Koding VM right from a browser tab with a full-page terminal application. I have also heard great things about nitrous.io as an alternative, however I have not personally tried it myself.

Option 5: Windows on Chrome OS

Description: I have not tried this yet but I have heard this is possible and in the works. Expect more on this topic soon. Time to Install: Unknown How: If you are feeling brave, try this oldie but goodie: http://gizmodo.com/5257386/how-to-install-windows-7-on-almost-any-netbook Here's another: http://www.sevacall.com/blog/2013/04/s/computer-repair/install-windows-on-chromebook/
